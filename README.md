# Advanced OSINT Suite (Python)

Ye ek educational OSINT (Open-Source Intelligence) tool hai jo publicly available information collect karne ke liye banaya gaya hai. Iska use sirf **legal & ethical** purpose ke liye karein.

## Features
- Username OSINT (GitHub, Instagram, Twitter, etc.)
- IP lookup (Location, ISP, org info)
- Domain WHOIS lookup
- Email OSINT (basic + MX records)
- Image EXIF metadata extractor

## Installation

```bash
git clone https://github.com/hppadhar/osint-suite.git
cd osint-suite
pip install -r requirements.txt
